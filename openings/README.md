# Chess Cheater Opening PGNs

Import any file here with `File > Import PGN File` and a relative path like:

`openings/01_ruy_lopez.pgn`

These are short, common opening starter lines intended as positions to explore with Stockfish.

More branch-specific lines are in `openings/variations/`, for example:

`openings/variations/01_ruy_lopez_berlin.pgn`
