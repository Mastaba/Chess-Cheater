# Chess Cheater Opening PGNs

Import any file here with `File > Import PGN File` and a relative path like:

`openings/01_ruy_lopez.pgn`

These are short, common opening starter lines intended as positions to explore with Stockfish.

More branch-specific lines are in `openings/variations/`, for example:

`openings/variations/01_ruy_lopez_berlin.pgn`

The curated established pack is in `openings/established/`. It adds 75 practical theory lines across common open games, Sicilians, French and Caro-Kann branches, modern defenses, 1.d4 systems, and major Indian defenses.

The larger extra study pack is in `openings/extra/`. It adds 175 more importable lines, including novelty lines such as Boncloud, Cow, Brandon Jacobson Gambit, The Drunk Magnus Switcharoo, and Knights In A Meadow. Those novelty entries include full setup lines plus several branches.
