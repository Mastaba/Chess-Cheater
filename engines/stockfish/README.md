Stockfish is bundled locally from the official Stockfish release download:

https://github.com/official-stockfish/Stockfish/releases/latest/download/stockfish-windows-x86-64.zip

The local extracted `stockfish/` folder includes the Windows engine executable,
source files, README, authorship information, and GPL license text
(`Copying.txt`). The source repository may omit the oversized executable from
git history; downloadable Windows release ZIPs include it.
Stockfish is licensed under GPL v3; keep the included source and license files
with the binary if this project is redistributed.
